5Causal

6Brief

形成可解释决策摘要

Publish

公开网站建议先用静态部署。

当前源码是纯静态 HTML/CSS/JS，可直接发布，不需要服务器和数据库。

GitHub Pages

最适合开源项目主页。把 `airmosaic-ai-web` 推到 GitHub 后，在仓库 Pages 设置里选择分支发布。

Vercel

适合后续迁移到 Next.js 或接入 API。可以连接 GitHub 仓库，也可以直接上传静态目录。

Cloudflare Pages

适合静态站和自定义域名，支持 Git 集成、直接上传和预览链接。
